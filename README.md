# Анализатор TF-IDF (тестовое Леста)

## Функционал 

* Загрузка текстового файла
* Извлечение слов из текста
* Расчет TF-IDF
* Отображение топ-50 слов

## Установка
```
git clone https://github.com/kun1ts4/tfidf.git
cd tfidf
go mod tidy
```

## Запуск
```
go run cmd/api/main.go
```
Откройте http://localhost:8080
